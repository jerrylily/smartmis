【Customer】
1. 客户类别 varchar4   custype (eg:1-个人 2-企业)
2. 客户性质 varchar16  CusProperty (eg:内宾散客,台籍散客...) 
3. 客户来源 varchar16  CusFrom (eg:客人介绍,去哪儿,长荣团...)
4. 结算方式 varchar4   PayType (eg:现结,月结...)
5. 证件类别 varchar16  PassType (eg:身份证,护照,台胞证...)
6. 电话类别 varchar4   PhoneType (eg:座机,手机...)
7. 地址类别 varchar4   AddressType (eg:送票,家庭,工作...)
8. 里程卡类别 varchar2 MilescardType (eg:CA,...)
